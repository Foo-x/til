# メモリ

- SLC (Single Level Cell)
    - 各セルに1ビットのデータを書き込めるメモリ
    - 高速・低消費電力・高耐久・高コスト
    - コストが高い
- MLC (Multi Level Cell)
    - 各セルに3ビット以上のデータを書き込めるメモリ
    - 低速・高消費電力・低耐久・低コスト
- メモリインタリーブ
    - メインメモリをバンクと呼ばれる複数の領域に分けて、並行にアクセスすることで高速化する技術
- DMA (Direct Memory Access)
    - CPUを介さずに、専用のコントローラとなるチップを介して入出力を行う技術
