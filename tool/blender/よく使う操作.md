# よく使う操作

「テンキーを模倣」はテンキーでない数字の操作が使えなくなるので設定しないほうが良い。  
Ctrl + Alt + Q による4分割表示や `` ` `` によるパイメニューなどで代替できる。  
キーボードが日本語配列で `` ` `` が押しにくい場合は、ビューメニューの「選択を表示」や「視点」を右クリックしてお気に入りツールに追加すれば、Qで呼び出せる。  
他のテンキー操作も同様。


## オブジェクト操作

- Shift + A
    - オブジェクトの追加
- X
    - 削除
- A
    - すべて選択
- Shift + クリック
    - 複数選択
- G
    - 移動
    - X/Y/Z で対応する軸に固定
    - Shift + X/Y/Z で対応する軸以外の軸に固定
    - 中クリックをドラッグして離すと軸の選択
    - Gで辺に固定
        - さらにCで辺の延長上にも移動可能
        - さらにEで整列
- R
    - 回転
    - X/Y/Z で対応する軸に固定
    - Shift + X/Y/Z で対応する軸以外の軸に固定
    - 中クリックをドラッグして離すと軸の選択
- S
    - 拡大縮小
    - X/Y/Z で対応する軸に固定
    - Shift + X/Y/Z で対応する軸以外の軸に固定
    - 中クリックをドラッグして離すと軸の選択
- Alt + G/R/S
    - それぞれ移動/回転/拡大縮小をリセット
- Shift + D
    - 複製
- Alt + D
    - リンク複製
- Esc
    - 各操作をキャンセル
- Ctrl + ドラッグ
    - スナップ
- Shift + ドラッグ
    - 高精度ドラッグ
- Shift + Tab
    - スナップ
- Ctrl + Shift + Tab
    - スナップ先メニュー
- O
    - プロポーショナル編集
    - 適用させたくない頂点/辺/面はHで隠しておくと良い
- オブジェクトメニュー → トランスフォーム → オブジェクトの整列
    - 選択した複数オブジェクトの整列
- Z
    - シェーディングのパイメニュー
- Shift + S
    - スナップのパイメニュー
- .
    - ピボットポイントのパイメニュー
- ,
    - 座標系のパイメニュー
- Ctrl + Tab
    - モードのパイメニュー
- Alt + W
    - フォールバックツールのパイメニュー
- M
    - 選択したものをコレクションに移動
- Shift + M
    - 選択したものをコレクションにリンク
- Ctrl + Alt + G
    - 選択したものをコレクションから除外
- Tab
    - オブジェクトモード/編集モードの切り替え
- Ctrl + A
    - 適用メニュー
- Shift + Z
    - ワイヤーフレーム
- 複数選択した状態で Ctrl + J
    - オブジェクトを結合
    - モディファイアが設定されている場合は最後に選択したものだけ残る
- 複数選択した状態で Ctrl + L
    - マテリアルやモディファイアなどを最後に選択したものにそろえる
- Ctrl + 1から5
    - サブディビジョンモディファイアを設定
    - 数字が大きいほど細かくなる


## メッシュ操作

オブジェクト操作のキーは一部メッシュに対しても使用可能。

- I
    - インセット（面の差し込み）
    - 複数選択している場合、さらにIでそれぞれに差し込み
- E
    - 押し出し
- Ctrl + 右クリック
    - マウスの位置に押し出し
- Alt + E
    - 押し出しメニュー
    - 法線に沿って押し出せる
        - 形が斜めになっているときはオペレータパネルから均一オフセットをオンにする
    - 多様体を押し出しにすると、立体の端の面を内側に押し出したときに、側面も追従させられる
- Ctrl + R
    - ループ切断
    - Eで隣接する辺の形状に合わせる（均一）
        - さらにFでもう片方の辺の形状に合わせる（反転）
    - スクロールで分割数を変更
    - Alt + スクロール で滑らかさを変更（スムーズ）
    - 位置を指定する際に右クリックで均等な位置になる
- K
    - ナイフ切断
    - Aで角度制限ができる
        - 続けて数値を入力すると角度の単位を指定できる
        - 最初は見た目上の角度だが、続けてAを入力すると最後のカット辺に対する角度になる
- Ctrl + X
    - 溶解
- Ctrl + B
    - 辺をベベル
- Ctrl + Shift + B
    - 頂点をベベル
- P
    - 分離
- F
    - 辺の選択時は面を張る
    - 頂点の選択時は辺を張るが、面に結合されないのでナイフ切断か頂点の連結で辺を作ったほうが良い
- J
    - 頂点の連結
- V
    - 切り裂き
- Shift + Alt + S
    - 球に変形
- オブジェクト上にマウスカーソルがある状態で Alt + Q
    - 編集するオブジェクトを切り替え
- 2つの面を選択した状態で辺メニューの「辺ループのブリッジ」
    - 2つの面の間をつなげる
- 複数選択した状態で選択メニューの「チェッカー選択解除」
    - 1つ飛ばしで選択
- 1
    - 頂点選択モード
- 2
    - 辺選択モード
- 3
    - 面選択モード
- Ctrl + V
    - 頂点メニュー
- Ctrl + E
    - 辺メニュー
- Ctrl + F
    - 面メニュー
- 右上の「自動頂点マージ」
    - 頂点が重なったら自動でマージ
    - 横のオプションからマージする距離の閾値を設定できる
- Ctrl + クリック
    - 間にあるものも含めて最短経路で選択
    - 複数経路がある場合はその中の1つ
- Ctrl + Shift + クリック
    - 間にあるものも含めて最短経路で選択
    - 複数経路がある場合はそれらすべて
- Alt + クリック
    - ループ選択
    - 縦か横かはクリックした位置によって決まる
- 辺選択モードで Ctrl + Alt + クリック
    - リング選択
- 面を選択した状態で Shift + N
    - 面の向きを外側にする
- 面を選択した状態で Ctrl + Shift + N
    - 面の向きを内側にする
- Ctrl + テンキーの+/-
    - 選択範囲を拡大/縮小
- Ctrl + Shift + テンキーの+/-
    - 選択範囲を次のアクティブに拡大/縮小
    - 同じ間隔で選択する機能
- Ctrl + L
    - つながっているものをすべて選択
    - 選択する範囲をオペレータパネルからマテリアルやシームで選べる
- Ctrl + I
    - 選択を反転
- Shift + G
    - 類似選択
    - 同じ平面上、マテリアル、長さ、方向などが同じものを選択できる
- M
    - マージ
    - すべての頂点を選択して「距離で」を選ぶと一括でマージできる
- Ctrl + Shift + Alt + S
    - 剪断
    - 歪ませる機能
- Shift + W
    - 曲げ
- 辺ループを選択した状態で面メニューからグリッドフィル
    - 辺ループ内を格子状に埋める
- 頂点選択モードで選択メニューの「特徴で全選択」から「非多様体」
    - 孤立した頂点を選択
- 辺選択モードで選択メニューの「特徴で全選択」から「非多様体」
    - 切り口を選択
- 選択メニューの「特徴で全選択」から「埋もれた面」
    - 埋もれた面を選択
- 頂点や辺を選択した状態でサイドバーのアイテムからクリースを設定
    - 設定した値の割合だけサブディビジョンがかからないようになる
- オブジェクトのプロパティで「ノーマル」の「自動スムーズ」にチェックを付け、辺を選択している状態で辺メニューの「シャープをマーク」
    - 自動スムーズに設定した角度より大きいものにスムーズシェードがかからないようにする
    - 180度にすることが多い
- 面メニューの「交差（ナイフ）」
    - 面同士の交差している箇所に辺を追加


## 視点操作

- テンキーの1
    - 正面からの視点に移動
- テンキーの3
    - 右からの視点に移動
- テンキーの7
    - 上からの視点に移動
- テンキーの9
    - 現在の視点の裏側に移動
- テンキーの0
    - カメラからの視点に移動
- テンキーの2
    - 視点を15度下からに移動
- テンキーの4
    - 視点を15度左からに移動
- テンキーの6
    - 視点を15度右からに移動
- テンキーの8
    - 視点を15度上からに移動
- 中ドラッグ
    - 視点回転
- Shift + 中ドラッグ
    - 視点移動
- 5
    - 透視投影と平行投影の切り替え
- Home
    - 全体を表示するように視点を合わせる
- テンキーの.
    - 選択しているオブジェクトに視点を合わせる
- `
    - ビューのパイメニュー
- Ctrl + Alt + 0
    - 現在の視点をカメラに設定する
- サイドバーのビューから「カメラをビューに」
    - カメラ視点のときに視点の移動に合わせてカメラを移動する
- カメラのプロパティから「ビューポート表示」の「コンポジションガイド」
    - カメラビュー時にガイドを表示


## 表示

- Alt + Z
    - 透過表示
- /
    - 選択したオブジェクトのみを表示
- H
    - 選択したものを非表示
- Shift + H
    - 選択したもの以外を非表示
- Alt + H
    - すべて表示
- 1から0
    - 対応するコレクション以外を非表示
- Shift + 1から0
    - 対応するコレクションの表示/非表示
- 右上のシェーディングのドロップダウンでカラーをランダムにする
    - 各オブジェクトをランダムに着色
- 右上のシェーディングのドロップダウンでキャビティをオンにする
    - 真上から見たときも段差を確認
- 右上のビューポートオーバーレイから「面の向き」
    - 面の向きを表示
- 右上のビューポートオーバーレイから「ノーマル」
    - 法線を表示
- Alt + B
    - クリップ表示
    - 選択したボックス内のみ表示する


## ウィンドウ操作

- Ctrl + Alt + Q
    - 4分割表示
- Ctrl + Space
    - ビューポートの最大化
- ビューポートの4隅からドラッグ
    - 分割表示/削除
- ビューポートの4隅から Shift + ドラッグ
    - そのビューポートを新しいウィンドウに分離
- T
    - ツールバーの表示
- N
    - サイドバーの表示
- Shift + Space
    - ツールバーメニュー


## その他

- F12
    - レンダリング
- Alt + S
    - テクスチャ画像/レンダリング画像の保存
- F9
    - オペレータパネルを再表示
- アウトライナーでモディファイアを移動
    - 同じオブジェクト内であれば適用順を変更
    - 違うオブジェクトであればコピー
    - モディファイアのルートを違うオブジェクトに移動するとリンク
- アウトライナーで右上のフィルタから選択可能を追加
    - 追加されたアイコンをクリックしてオフにするとオブジェクトをロック
- Shift + R
    - 最後の操作を繰り返す
    - たとえばオブジェクトを複製して3m右に移動させたあとに実行すると、さらに3m右にもう1つオブジェクトができる
- アウトライナーやプロパティなどで折り畳みの三角を上下にドラッグ
    - 折り畳みを一括で開く/閉じる
