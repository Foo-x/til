# PFD

Process Flow Diagram の略で、プロセスの流れを図示する。  
Data Flow Diagram (DFD) をベースにしていて書式はほとんど同じ。  
基本はプロセスを丸、成果物を四角で書いて矢印でつなげるだけ。
