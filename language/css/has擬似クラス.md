# has擬似クラス

`:has()` 擬似クラスは受け取ったセレクタにマッチする子要素を持つ要素にマッチするセレクタとして働く。  
たとえばカードコンポーネントが画像を持つかどうかや、チェックボックスのいずれかがチェックされているかどうかでスタイルを変えたりできる。
