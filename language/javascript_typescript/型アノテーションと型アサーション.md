# 型アノテーションと型アサーション

```ts
type A = {
  foo: string;
};

// 型アノテーション
const a1: A = {
  foo: '',
};

// 型アサーション
const a2 = {
  foo: '',
} as A;
```

型アノテーションはプロパティに過不足があるときにエラーになり、より安全なのでこちらを使う。  
なお、プロパティの型が異なるときはどちらもエラーになる。

```ts
type A = {
  foo: string;
};

// プロパティが不足しているとき

// エラーになる
const a1: A = {};

// エラーにならない
const a2 = {} as A;

// プロパティが過多なとき

// エラーになる
const a3: A = {
  foo: '',
  bar: '',
};

// エラーにならない
const a4 = {
  foo: '',
  bar: '',
} as A;

// プロパティの型が異なるとき

// エラーになる
const a5: A = {
  foo: 1,
};

// エラーになる
const a6 = {
  foo: 1,
} as A;
```
