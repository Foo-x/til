# シェルコマンドと連携する

- カーソル位置の行を入力として渡し、出力で置換
    - ノーマルモードから `!!` で `:.!` になる
    - `:.!command`
- 選択範囲を入力として渡し、出力で置換
    - ビジュアルモードから `!` で `:'<,'>!` になる
    - `:'<,'>!command`
- 入力を渡さずにカーソル位置の下の行に挿入
    - `:r !command`
    - `:r` 後の空白は省略可
- ファイル全体を入力として渡して実行
    - `:w !command`
    - `:w` 後の空白は省略不可
        - `:w!` はファイル名を指定して上書きする別のコマンド

使い方例

- 編集中の JavaScript ファイルを実行
    1.  `:w !node`
- 一括リネーム
    1. `:r!ls foo_*.txt` でファイル一覧を挿入
    2. 矩形編集で各行を `mv` でリネームするコマンドに変更
    3. `:w !sh` で実行
