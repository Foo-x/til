# データ構造と各処理の計算量

## 可変長配列

| 処理 | 計算量 |
| --- | --- |
| 長さの取得 | 1 |
| i番目の取得 | 1 |
| i番目の更新 | 1 |
| 末尾への追加 | 1 |
| 末尾以外への追加 | N |
| i番目の削除（順序保存） | N |
| i番目の削除（順序非保存） | 1 |
| 末尾の削除 | 1 |
| 検索 | N |
| 最小値の取得 | N |
| 最大値の取得 | N |
| スライス | k |
| ソート | NlogN |

## 集合

| 処理 | 計算量 |
| --- | --- |
| 長さの取得 | 1 |
| 追加 | 1 |
| 削除 | 1 |
| 検索 | 1 |
| union S+T | len(S) + len(T) |
| intersection S*T | min(len(S),len(T)) |
| difference S-T | len(S) |
