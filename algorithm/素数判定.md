# 素数判定

- 決定的
    - AKS法
        - O(log(n^7.5)*log^k(log(n^7.5)))
            - kは任意の定数
        - 次数が高いので最速ではない
    - APR法
        - (log(n))^(O(log^3(n)))
    - エラトステネスの篩
        - O(n*log^2(n))
        - n以下のすべての素数を見つける
    - アトキンの篩
        - O(n/log^2(n))
        - エラトステネスの篩の改良版
- 確率的
    - Miller-Rabin法
        - O(k*log^3(n))
            - kは素数判定を行う回数で、4^(-k)の確率で偽陽性となる。（偽陰性はない）
